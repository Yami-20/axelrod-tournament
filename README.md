# ◈ Axelrod's Tournament

An interactive simulation of Robert Axelrod’s famous Iterated Prisoner's Dilemma tournament.

This project demonstrates how cooperation can emerge between competing strategies over repeated interactions.

---

## What You Can Do

- **Explore 14 strategies** from the original Axelrod tournament
- **Run 1v1 duels** and see move-by-move outcomes
- **Simulate full tournaments** with rankings and charts
- **Build your own strategy** and test it against others

---

## Why It Matters

This simulation shows how simple strategies like **Tit for Tat** can outperform aggressive ones, proving that cooperation can evolve even in competitive environments.

---

## The Core Idea

In the Prisoner's Dilemma:

- Cooperation can lead to mutual benefit
- Defection can give short-term advantage
- Repeated interaction changes everything
